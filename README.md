# 🚀 Bikku's TradeLab

A **gamified trading simulator platform** where users can practice trading, complete challenges, earn rewards, and level up like a real trading game.

---

## 🌟 Features

### 🎮 Core Trading System
- Virtual trading (Buy / Sell simulation)
- Profit & loss tracking
- Trade history system
- Live market simulation

---

### 🏆 Funded Challenge Mode
- Profit target system (+10%)
- Daily loss limit (-5%)
- Challenge pass/fail system
- Separate trading mode experience

---

### 🪙 CRYPT Economy System
- In-app currency: **CRYPT (Ȼ)**
- Earn CRYPT from trades and missions
- Spend CRYPT on rewards and upgrades

---

### ⭐ Progress System
- XP system for every trade
- Level up system
- Feature unlock progression

---

### 🎰 Engagement Features
- Spin wheel reward system
- Daily missions
- Login progress saving (streak concept)
- Achievements & badges

---

### 📊 UI & Experience
- Modern dark trading interface
- Glassmorphism UI design
- Mobile-friendly layout
- Smooth animations & feedback system
